# รูปสินค้า (Product images)

รูปสินค้าถูกอัปโหลดไปที่ **Cloudinary** ไม่เก็บในโฟลเดอร์นี้

- ใช้ปุ่ม "อัปโหลดรูปสินค้า" ในฟอร์มเพิ่ม/แก้ไขสินค้า → ระบบจะอัปโหลดไป Cloudinary และเก็บ URL ไว้ในสินค้า
- รองรับ: jpg, jpeg, png, webp ขนาดไม่เกิน 5MB
- โฟลเดอร์นี้ (`client/public/products/`) ไม่ใช้สำหรับเก็บรูปสินค้าแล้ว
